# Fontes de Dados

## Dataset Principal (MVP)

**Dataset sintético gerado pelo ViaStat**
- Seed: 42 (reproduzível)
- 10 rotas, 7 dias, ~250k+ pings
- Anomalias controladas
- Formato: CSV + GeoJSON

## Fontes Públicas Potenciais (pós-MVP)

| Fonte | Descrição | Status |
|-------|-----------|--------|
| OpenBus SP | Dados históricos de GPS de ônibus de São Paulo | A avaliar |
| GTFS-RT SP | Arquivos estáticos de posição em tempo real | A avaliar |
| SPTrans Dados Abertos | Dados de transporte público de SP | A avaliar |
| EMTU | Dados de transporte metropolitano | A avaliar |

## Critérios para Inclusão de Fonte Pública

1. Licença permite uso acadêmico/portfólio
2. Documentação de schema disponível
3. Dados históricos acessíveis (não só tempo real)
4. Recorte pequeno pode ser validado manualmente
